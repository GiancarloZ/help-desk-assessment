## Installation

clone repo

## To start the node server

```
cd help-desk-assessment-server
run npm install && npm run dev
```

## To spin up frontend environment

```
cd help-desk-assessment-client
run npm install && npm run dev
```

## Usage

Submit tickets. View and edit ticket.
